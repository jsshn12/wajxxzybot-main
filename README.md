### Cara Installnya
Sebelum lu jalanin sc nya install dulu lah.
```bash
> kalo lu belum punya apk termux, download di playstore
> masuk ke apk termux lalu ketik dibawah ini!
> git clone git remote add origin https://github.com/jsshn12/cyberoren.git
git branch -M main
git push -u origin main
> cd botst4rz
> bash install.sh
> node index.js
> Tinggal scan qr dah
```
